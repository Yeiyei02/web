Proyectos Destacados

Aquí puedes encontrar la documentación de tres proyectos que te ayudarán a practicar y aprender más sobre programación:

Aplicación de Tareas

Esta aplicación permite a los usuarios gestionar sus tareas diarias de manera eficiente. Los usuarios pueden agregar, editar y eliminar tareas, así como marcar las que han completado. Utiliza HTML, CSS y JavaScript para una experiencia de usuario interactiva y fluida.

Juego de Adivinanza

Un juego simple donde los jugadores deben adivinar un número generado aleatoriamente por la computadora. El juego proporciona pistas sobre si el número ingresado es demasiado alto o bajo. Este proyecto utiliza JavaScript para la lógica del juego y CSS para un diseño atractivo.

Portafolio Personal

Un sitio web personal que muestra los proyectos y habilidades del desarrollador. Incluye secciones sobre experiencia, educación y contacto. Este proyecto es una excelente manera de presentar el trabajo de uno mismo y se construye utilizando HTML, CSS y JavaScript para la interactividad.
